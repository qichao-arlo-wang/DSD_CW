\section{Scope and Objectives}
% =============================================================

This report covers Task 7 (7a/7b/7c) and Task 8 of the coursework.
The target function is
\[
F(x)=\sum_i f(x_i),\quad f(x)=0.5x + x^3\cos\left(\frac{x-128}{128}\right)
\]
under single-precision floating-point representation.

\textbf{What this report demonstrates:}
\begin{itemize}
    \item Task 7a: CORDIC error analysis and parameter selection using Monte Carlo.
    \item Task 7b: CORDIC architecture choices, latency/timing/resource trade-offs.
    \item Task 7c: Step-2 and Step-3 hardware architectures, integration and performance.
    \item Task 8: Throughput-oriented final architecture and comparison to Task 7 best case.
\end{itemize}
